Docker 🐋

En este note vas a encontrar una lista de los comandos mas utilizados para trabajar con Docker

📋🚚 Listar todos los contenedores (docker ps [OPTIONS])

docker ps -a


🚮🚚 Borrar todos los contenedores (docker container COMMAND)

docker container prune


⚒🧱 Construir imagen (docker build [OPTIONS] pathDockerfile)

docker build -t image-name .


▶🚚 Correr un contenedor (docker run [OPTIONS] IMAGE [COMMAND] [ARG...])

docker run -d --rm -p 80:5000 --name container-name image-name

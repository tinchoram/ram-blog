Python

En este note vas a encontrar los comandos más utilizados para trabajar con Python 🐍😍


📁 Crear entorno virtual (python -m venv nombre_venv)

python -m venv venv


📁 Activar entorno virtual

Windows: .\venv\Scripts\activate

Linux o Mac: source venv/bin/activate


📁 Desactivar entorno virtual

deactivate


📁 Instalar dependencias con pip (Package Installer for Python): pip install module_name

pip install pandas


📁 Listar paquetes instalados en Python

pip freeze


📄 Crear requirements.txt

pip freeze > requirements.txt


📄 Instalar requirements.txt

pip install -r requirements.txt
